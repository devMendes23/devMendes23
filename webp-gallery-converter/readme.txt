Converte as imagens JPG/PNG da galeria do WordPress para WebP, reduzindo o peso das páginas.

== Description ==

O WebP Gallery Converter gera versões WebP das suas imagens e as entrega
automaticamente para navegadores compatíveis, deixando o site mais rápido.

Funcionalidades:

* Conversão automática no upload (imagem original + miniaturas).
* Conversão em massa das imagens já existentes na biblioteca.
* Entrega de WebP no front-end apenas para navegadores compatíveis.
* Configuração de qualidade e opção de manter os arquivos originais.
* Suporte a GD e Imagick.

== Installation ==

1. Copie a pasta `webp-gallery-converter` para `wp-content/plugins/`.
2. Ative o plugin no menu "Plugins" do WordPress.
3. Acesse "Mídia > WebP Converter" para ajustar as configurações.
4. Use o botão "Iniciar conversão em massa" para converter as imagens antigas.

== Frequently Asked Questions ==

= Meu servidor precisa de quê? =
Da extensão GD (com suporte a WebP) OU da extensão Imagick no PHP.

= Os arquivos originais são apagados? =
Não por padrão. Você pode alterar isso nas configurações.
